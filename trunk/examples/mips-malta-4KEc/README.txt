Компилятор
==========
    Я использовал компилятор GCC 4.7.3 своей сборки. Бинарные версии под Linux (Ubuntu 13.04) и Windows выложены на сайте uOS (главная страница: http://code.google.com/p/uos-embedded/)
    
    1) Установка под Linux.
    Скачать с сайта проекта компилятор GCC 4.7.3 для MIPS, разархивировать и скопировать с правами суперпользователя в каталог /usr/local
    
    2) Установка под Windows.
    Сборка под Windows осуществляется с использованием MinGW/Msys. Их тоже можно скачать с сайта проекта. Далее скачать компилятор GCC 4.7.3 для MIPS (windows binaries), разархивировать и скопировать в MSys/1.0/local
    
Сборка примеров
===============
    1) Скачать из SVN наиболее свежую версию uOS (http://code.google.com/p/uos-embedded/source/checkout)
    2) Под Linux запустить эмулятор терминала. Если используется Windows, то запустить Msys.bat.
    3) Перейти в папку примеров для MIPS MALTA. Под MinGW диски Windows обозначаются в пути как /c, /d и т.д.
    Папка с примерами лежит в uos-embedded/examples/mips-malta-4KEc
    4) Собрать примеры командой make.
    
Загрузка и запуск примеров
==========================
    Для загрузки примеров на плату я пользовался только средствами YAMON и только UART.
    1) Подключить UART0 и UART2 (который подключается через специальную планку) к ПК. Планка предназначена только для MALTA (специально для неё перепаяна), нельзя подключать её к обычному ПК!
    2) Запустить 2 копии программы-эмулятора терминала. Я использовал minicom и kermit под Linux. Под Windows можно использовать, например, putty. Подключение по UART0 настроить на 38400 8N1 с аппаратным контролем потока, UART1 - 115200 8N1 с аппаратным контролем.
    3) Включить питание платы. Если всё подключено и настроено правильно, то в терминале UART0 YAMON сообщит о конфигурации платы и выдаст приглашение.
    4) Для загрузки необходимо дать команду YAMON:
        load asc:
       Далее необходимо отправить один из SREC-файлов как текстовый (в minicom делается командой CTRL-A S).
    5) Для запуска примера дать команду YAMON
        go
        
       Сообщения uOS принимаются по UART2.
