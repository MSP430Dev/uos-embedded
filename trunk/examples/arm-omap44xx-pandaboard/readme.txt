Сборка и загрузка тестовых проектов в Pandaboard
------------------------------------------------
Сборка всех тестовых примеров осуществляется командой make. Результатом этой
операции будет создание бинарный файлов для всех тестовых проектов, пригодных
для запуска на Pandaboard из флеш-карты формата SD.
Далее нужный бинарный файл нужно переименовать в MLO и скопировать в корень 
флеш-карты. Флеш-карта должна иметь файловую систему FAT (FAT16, FAT32).
Бинарный файл для проекта, указанного в переменной TEST в Makefile, 
автоматически будет скопирован в файл MLO.
